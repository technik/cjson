cjson::Json::operator bool ( ) const
explicit

Cast to boolean This will produce different results depending on the actual type of data stored in the json. A boolean type (true or false), will be retrieved as such. A number will return false for 0 and true otherwise. A null element will always return false. An array or object will return false if empty, true otherwise.

Returns
the value of this json as a boolean element.
